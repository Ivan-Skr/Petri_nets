Инструкция к симулятору

1. Выбрать объект для создания и нажать на соответствующую кнопку.

Для смены выбора нажмите на другую кнопку.

2. Нажмите на область, в которую хотите поместить объект.

Для расположения дуги необходимо нажать дважды: в начало и конец

дуги. Границы дуг должны находиться внутри какого-либо объекта.

3. Для запуска сети нажмите стрелку вправо (→). Это запустит

первую итерацию. Для следущей итерации нажмите стрелку еще раз.

Количество возможных итераций равно количеству переходов.

Когда итерации закончатся, экран очистится, и вы сможете создать

ещё одну сеть Петри.

Для очистки экрана нажмите клавишу "R" на клавиатуре.
